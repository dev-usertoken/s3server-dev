How to use the example
======================================
1. Copy the content of simulator_manifest.json (JSON file) to ```kinetic-java-tools/kinetic-console/config/default/hwview_default.json```

2. cd kinetic-console 

3. mvn clean package

4. cd bin
   
5. sh startExampleSimulator.sh
   
6. sh startKineticConsole.sh

7. open the URL "http://localhost:8080/kinetic/console" with a browser
